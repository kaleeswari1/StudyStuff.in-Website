# 📚 StudyStuff.in

> **A simple and responsive stationery & study materials website built using HTML and CSS.**

---

# 🌟 Overview

**StudyStuff.in** is a beginner-friendly static website designed to showcase a collection of study materials and stationery products for students, schools, and offices. The website features an organized layout using **CSS Grid**, making navigation simple and the content easy to explore.

This project demonstrates the use of HTML5 semantic elements and CSS for creating a clean, structured, and user-friendly webpage.

---

# 🎯 Project Objectives

* 📖 Create an attractive educational website.
* 🎒 Showcase stationery and study products.
* 🧭 Provide easy website navigation.
* 🖼️ Display a gallery of study materials.

# 🎨 Website Layout

The website uses **CSS Grid Layout** with the following structure:

```text
+--------------------------------------+
|              Header                  |
+--------------------------------------+
|            Navigation                |
+--------------------------------------+
| Sidebar |      Main Content          |
|         |   Gallery & Contact        |
+--------------------------------------+
|              Footer                  |
+--------------------------------------+
```
